## 任务

- 以你的姓名+学号作为标题进行发布开启一个 issue，内容随便。

- 你可以拿这个仓库做任何尝试，包括添加文件，修改文件等，你也可以在 Issues 中与其他人互动

- 注册 GitHub 账号
- 使用 Git 命令
- 遇到文件冲突
- 网络问题
